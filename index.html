📚 과목 관리

과목을 추가하거나 삭제합니다. 이름은 자유롭게 입력하세요.
예: 국어_기본, 국어_심화, 영어문법, 수학1 등

※ 과목 삭제 시 해당 과목의 모든 데이터가 삭제됩니다.

📁 과목 그룹 관리

수학A·수학B처럼 별도 탭으로 운영하는 과목을 하나의 그룹으로 묶으면, 교재 관리 화면에서 나란히 보이고 합산 통계도 함께 확인할 수 있습니다.

📅 과목별 수강 요일 · 자동 배정

과목별로 수강 요일과 시작일을 설정하면 수강 예정일을 자동으로 채워줍니다. 이미 입력된 날짜는 덮어쓰지 않습니다.

※ 자동 채우기: 비어있는 날짜만 채웁니다.  전체 덮어쓰기: 기존 날짜도 모두 새로 계산합니다.

📖 과목별 교재 · 강의 범위 설정

교재명과 강의 범위를 입력하면 행이 자동으로 생성됩니다.
📌 교재 탭 자동 분기: 첫 교재는 기존 탭 이름이 변경되고, 두 번째 교재부터 새 탭이 생성됩니다.
예) 수학 탭에 "기본서" 추가 → 탭: 수학-기본서  |  "심화서" 추가 → 탭: 수학-기본서 + 수학-심화서

📅 시험일 설정

시험일을 등록하면 3회독 완료 예정일이 시험 이후인 강의에 경고 표시가 나타납니다.

🐾 공부 파트너 설정

파트너 이름을 설정하면 오늘 탭 브리핑에서 사용됩니다.

이름 설정 (최대 8자)

📆 과목별 수강 마감 설정

시험일 기준으로 과목별 수강 완료 마감을 설정합니다.
역사 = 0 (시험 당일까지), 나머지 = -14 (시험 2주 전까지)

※ 음수 = 시험일 이전 N일. 0 = 시험 당일까지. 역사처럼 시험 당일까지 수강 가능한 과목은 0으로 설정하세요.

🤖 AI 파트너 연동 (OpenRouter)

API 키를 입력하면 오른쪽 하단 채팅 버튼으로 파트너와 실시간 대화가 가능합니다.
무료 모델(Gemini 2.0 Flash) 사용 — openrouter.ai에서 무료 발급 가능

미연결
💡 연결이 안 될 때 대안 보기

openrouter.ai/keys → 가입 → Create Key → 복사 후 위 입력창에 붙여넣기
② Google AI Studio (Gemini 직접 연동)
aistudio.google.com → API 키 발급 (완전 무료)
→ 아래 "Gemini 직접 연동" 탭에서 입력

⚠ API 키는 이 기기의 브라우저에만 저장됩니다. HTML 파일에는 저장되지 않습니다.

⚖️ 하루 최대 수강 건수 제한

과부하 날짜가 생기면 초과 분을 다음 수강 요일로 자동 분산합니다.
예: 목요일에 13건 → 최대 5건 설정 시 8건을 다음 요일로 이동

건 / 일

※ 분산 실행은 미완료 수강 예정일만 대상입니다. 이미 완료된 강의는 건드리지 않습니다.

🎯 시험기간 집중 배정

시험 직전 특정 기간에 미완료 강의를 몰아서 수강해야 할 때 사용합니다.
선택한 과목의 미완료 강의를 지정 기간 내에 하루 N강씩 재압축 배정합니다.
이미 완료된 강의(수강 체크 완료)는 건드리지 않습니다.

강 / 일

※ 배정 실행 전 JSON 백업을 권장합니다. 실행 후 JSON 백업 → "일정만 병합" 으로 언제든 되돌릴 수 있습니다.

🎯 하루 목표 순공시간

타이머 링과 오늘 탭 순공시간 바에 적용되는 하루 목표 시간입니다.
예: 6으로 설정 시 6시간 채우면 링·바가 100%

시간 / 일  (기본값 8)
💡 현재 목표: 하루 8시간

🌙 새벽 공부 유예 시간

자정이 넘어도 이 시각 이전까지는 전날로 인정합니다.
예: 3으로 설정 시 새벽 0~2시 59분은 전날 날짜로 처리 → "내일로" 누르기 전 자정이 넘어도 지연으로 잡히지 않습니다.

시 이전까지 전날로 인정  (0 = 유예 없음, 최대 6)
💡 현재: 새벽 3시 이전은 전날로 인정합니다.

📌 수행 할일 — 마감일 설정 방식

오늘 탭 하단 추가 할일에서 유형을 수행으로 선택하면 마감일 날짜 피커가 나타납니다.
• 당일 버튼 클릭 → 오늘 하루만 표시 (일반 추가와 동일)
• 날짜를 6월 10일로 선택 → 6월 10일이 지나고 새벽 공부 유예 시간이 지날 때까지 매일 오늘 탭에 표시
• 체크 완료해도 마감일까지는 목록에 유지 (투명 처리)

💡 수행 항목은 별도 이월 팝업 없이 마감일 안에서 자동으로 오늘 탭에 표시됩니다.

🔗 회독 범위 합산 (묶음 복습)

회독이 쌓일수록 건수가 너무 많아지는 문제를 해결합니다.
2회독은 N강씩 묶어 같은 날 한꺼번에 복습하고, 3회독은 M강씩 묶어 배정합니다.
예: 2회독 2강 묶음 → 1·2강이 같은 날 / 3·4강이 같은 날 / …
1로 설정하면 기존처럼 강의별 개별 배정됩니다. 새로 수강 완료하는 강의부터 적용됩니다.
