자동저장

📚 과목 관리

과목을 추가하거나 삭제합니다. 이름은 자유롭게 입력하세요.
예: 국어_기본, 국어_심화, 영어문법, 수학1 등

※ 과목 삭제 시 해당 과목의 모든 데이터가 삭제됩니다.

📁 과목 그룹 관리

수학A·수학B처럼 별도 탭으로 운영하는 과목을 하나의 그룹으로 묶으면, 교재 관리 화면에서 나란히 보이고 합산 통계도 함께 확인할 수 있습니다.

시험일 설정

시험일을 등록하면 3회독 완료 예정일이 시험 이후인 강의에 경고 표시가 나타납니다.

⚖️ 하루 최대 수강 건수 제한

과부하 날짜가 생기면 초과 분을 다음 수강 요일로 자동 분산합니다.
예: 목요일에 13건 → 최대 5건 설정 시 8건을 다음 요일로 이동

건 / 일

※ 분산 실행은 미완료 수강 예정일만 대상입니다. 이미 완료된 강의는 건드리지 않습니다.

🎯 시험기간 집중 배정

시험 직전 특정 기간에 미완료 강의를 몰아서 수강해야 할 때 사용합니다.
선택한 과목의 미완료 강의를 지정 기간 내에 하루 N강씩 재압축 배정합니다.
이미 완료된 강의(수강 체크 완료)는 건드리지 않습니다.

강 / 일

※ 배정 실행 전 JSON 백업을 권장합니다. 실행 후 JSON 백업 → "일정만 병합" 으로 언제든 되돌릴 수 있습니다.

📅 과목별 수강 요일 · 자동 배정

과목별로 수강 요일과 시작일을 설정하면 수강 예정일을 자동으로 채워줍니다. 이미 입력된 날짜는 덮어쓰지 않습니다.

※ 자동 채우기: 비어있는 날짜만 채웁니다.  전체 덮어쓰기: 기존 날짜도 모두 새로 계산합니다.  초기화: 모든 수강 예정일을 삭제 후 다시 시작.

📖 과목별 교재 · 강의 범위 설정

교재명과 강의 범위를 입력하면 행이 자동으로 생성됩니다.
📌 교재 탭 자동 분기: 첫 교재는 기존 탭 이름이 변경되고, 두 번째 교재부터 새 탭이 생성됩니다.
예) 수학 탭에 "기본서" 추가 → 탭: 수학-기본서  |  "심화서" 추가 → 탭: 수학-기본서 + 수학-심화서

☁️ Firebase 실시간 동기화 연결됨 — 어느 기기에서 저장해도 자동으로 반영됩니다.
